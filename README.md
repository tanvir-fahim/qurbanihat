# QurbaniHat

QurbaniHat is a modern, transparent, and user-friendly online livestock marketplace designed to simplify the purchase and sale of sacrificial animals for Eid-ul-Adha.

## Purpose
The primary goal of QurbaniHat is to bridge the gap between farmers and buyers, providing a reliable platform where users can browse, compare, and connect with sellers to purchase healthy, high-quality animals without the hassle of traditional cattle markets.

## Key Features
- **Real-time Marketplace:** Browse a wide catalog of animals (cows, goats, bulls) with detailed specifications.
- **Secure Authentication:** User registration and login powered by Auth.js.
- **Interactive UI:** Dynamic animal cards with hover-triggered Lottie animations for an engaging experience.
- **Responsive Design:** A mobile-first, clean interface built with Tailwind CSS and DaisyUI.
- **Active Navigation:** Persistent navigation with active route highlighting and user session management.
- **Profile Management:** Users can manage their profiles and track their activity.

## Tech Stack & Dependencies
This project is built using **Next.js 14+ (App Router)** and utilizes the following packages:

### Core Dependencies:
- **`next`**: The React framework for the web.
- **`react` & `react-dom`**: Frontend library.
- **`lottie-react`**: For lightweight, performant vector animations.
- **`react-icons`**: For consistent iconography throughout the app.
- **`framer-motion`**: (Optional/if applicable) For smooth transitions.
- **`daisyui`**: Tailwind CSS component library for polished UI components.
- **`react-toastify`**: For elegant user notifications.
- **`better-auth`**: For authentication.

### Dev Dependencies:
- **`tailwindcss`**: Utility-first CSS framework.
- **`eslint`**: For maintaining code quality.

## Getting Started
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `npm install`
3. Configure your `.env` file with necessary API keys and database credentials.
4. Run the development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000) in your browser.
